eduroam 접속 안내

eduroam CAT 도구를 사용하면 클릭 몇 번으로 자동 설정이 완료됩니다.
수동 설정이 필요한 경우 아래 OS별 안내를 참고해 주세요.

권장
eduroam CAT (자동 설정)

eduroam CAT(Configuration Assistant Tool)을 사용하면 기기 설정을 자동으로 완료할 수 있습니다. Windows, macOS, iOS, Android 모두 지원합니다.

  • cat.eduroam.org에 접속하여 {{ site.t[page.lang].org }}를 검색합니다.
  • 운영체제에 맞는 설치 프로그램(프로파일)을 다운로드합니다.
  • 설치 후 계정(아이디@{{ site.realm }})과 비밀번호를 입력합니다.

CAT 도구를 사용하면 인증서 설정이 자동으로 처리됩니다.

  1. 1설정 > Wi-Fi에서 eduroam을 선택합니다.
  2. 2사용자 이름: 아이디@{{ site.realm }}, 비밀번호를 입력합니다.
  3. 3인증서 확인 화면에서 신뢰를 탭합니다.

iOS 15 이상: CAT 프로파일 설치를 권장합니다.

  1. 1Wi-Fi 목록에서 eduroam을 선택합니다.
  2. 2EAP 방식: PEAP, 2단계 인증: MSCHAPV2를 선택합니다.
  3. 3CA 인증서: 시스템 기본 인증서 사용 또는 다운로드한 인증서를 선택합니다.
  4. 4도메인: {{ site.realm }}, 사용자 이름: 아이디@{{ site.realm }}, 비밀번호를 입력합니다.

Android 11 이상: 도메인 필드 입력이 필수입니다.

인증서 정보

OS 신뢰 저장소에 CA 인증서가 설치된 경우, 인증서 신뢰 확인 창이 나타나지 않으며 인증서 갱신 시 별도 조치가 필요 없습니다.
CA 인증서 없이 수동 설정(TOFU)한 경우, 서버 인증서 만료 시 Wi-Fi 프로파일을 삭제하고 재연결해야 합니다.
TOFU(Trust On First Use): 최초 접속 시 사용자가 직접 서버 인증서를 확인하고 신뢰하는 방식입니다.
