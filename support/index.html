자주 묻는 질문 (FAQ)
처음 eduroam을 사용하려면 어떻게 하나요?
접속 방법 페이지에서 eduroam CAT 도구를 설치하는 방법을 안내합니다. CAT을 사용하면 기기별 수동 설정 없이 몇 번의 클릭으로 완료할 수 있습니다.
비밀번호를 잊었어요.
기관 통합 계정 포털 또는 정보통신팀({{ site.support.email }})으로 연락하시면 비밀번호 재설정을 도와드립니다.
잘 되던 eduroam이 갑자기 인증 오류가 납니다.
첫 연결에 TOFU(Trust On First Use, 첫 사용에 대한 신뢰)로 접속하셨다면 서버 인증서가 갱신됐을 수 있습니다. Wi-Fi 설정에서 eduroam 프로파일을 삭제하고 다시 접속하면 새 인증서로 재설정됩니다. 서버 인증서 유효기간은 접속 방법 페이지에서 확인하세요.
접속은 되는데 인터넷이 안 됩니다.
Wi-Fi 설정에서 eduroam을 삭제 후 재연결해 보세요. 문제가 지속되면 IP 자동 할당 설정을 확인하거나 헬프데스크로 문의해 주세요.
인증서 오류가 발생합니다.
eduroam CAT 프로파일을 재설치하면 인증서가 자동으로 갱신됩니다. 수동 설정 시 CA 인증서를 직접 설치해야 합니다. 접속 방법을 다시 확인해 주세요.
해외에서 eduroam에 접속하려면?
국내에서 CAT 프로파일을 설치해 두면 해외 eduroam AP에서 별도 설정 없이 자동 접속됩니다. 이용 지역 페이지에서 글로벌 커버리지를 확인하세요.
스마트폰과 노트북을 모두 연결할 수 있나요?
네, 동일한 계정으로 여러 기기를 동시에 연결할 수 있습니다. 각 기기에 CAT 프로파일을 설치하거나 수동 설정을 완료하세요.
