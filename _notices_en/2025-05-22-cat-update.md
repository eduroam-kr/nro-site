The eduroam CAT (Configuration Assistant Tool) profile has been updated.

## Affected Devices

All devices with a previously installed CAT profile

## What's Changed

- Updated server certificate information
- Improved compatibility with the latest OS versions (iOS 17.5, Android 14, Windows 11 24H2)

## Action Required

Please download and reinstall the latest profile from [cat.eduroam.org](https://cat.eduroam.org).  
It is recommended to remove the existing profile before reinstalling.

If your current profile is working normally, no action is required.
