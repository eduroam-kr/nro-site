eduroam Connection Guide

Use the eduroam CAT tool to configure your device automatically in just a few clicks.
For manual setup, refer to the per-OS instructions below.

RECOMMENDED
eduroam CAT (Auto Setup)

Use the eduroam CAT (Configuration Assistant Tool) to automatically configure your device. Supports Windows, macOS, iOS, and Android.

  • Go to cat.eduroam.org and search for {{ site.t[page.lang].org }}.
  • Download the installer for your operating system.
  • Run the installer and enter your account (username@{{ site.realm }}) and password when prompted.
  • Once installed, select eduroam from your Wi-Fi list to connect.
Go to cat.eduroam.org

This service is only available if your institution's administrator has registered with eduroam CAT.

Certificate Information

If you have the CA certificate installed in the OS trust store, no trust confirmation dialog will appear, and certificate renewals are handled automatically — no action needed.
If you set up manually without the CA certificate (TOFU), you must delete the Wi-Fi profile and reconnect when the server certificate expires.
TOFU (Trust On First Use): on first connection, the user manually inspects and trusts the server certificate.
