eduroam CAT (Recommended)

Use the eduroam CAT (Configuration Assistant Tool) to automatically configure your device in just a few clicks.

  1. 1 Go to cat.eduroam.org and search for {{ site.t[page.lang].org }}.
  2. 2 Download the installer for your operating system.
  3. 3 Run the installer and enter your institutional account (username@{{ site.realm }}) and password when prompted.
  4. 4 Once installed, select eduroam from your Wi-Fi list to connect automatically.

This service is only available if your institution's administrator has registered with eduroam CAT.

Using the CAT tool handles certificate setup automatically.

  1. 1Go to Settings > Wi-Fi and select eduroam.
  2. 2Username: username@{{ site.realm }}, then enter your password.
  3. 3Tap Trust on the certificate verification screen.

iOS 15 and later: installing a CAT profile is recommended.

  1. 1Select eduroam from the Wi-Fi list.
  2. 2EAP method: PEAP, Phase 2: MSCHAPV2.
  3. 3CA certificate: select Use system certificates or import the downloaded certificate.
  4. 4Domain: {{ site.realm }}, Identity: username@{{ site.realm }}, then enter your password.

Android 11+: the domain field is required.

Certificate Information

If you have the CA certificate installed in the OS trust store, no trust confirmation dialog will appear, and certificate renewals are handled automatically — no action needed.
If you set up manually without the CA certificate (TOFU), you must delete the Wi-Fi profile and reconnect when the server certificate expires.
TOFU (Trust On First Use): on first connection, the user manually inspects and trusts the server certificate.
