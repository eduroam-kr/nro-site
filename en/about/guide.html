  1. Home

Enrollment Steps Overview

A server administrator at the university or institution must apply directly.

01.
Register on eduroam.kr

The server administrator of the university or institution must register.

Go to registration site
02.
Administrator Approval

Once approved, the account gains university server administrator privileges.

03.
Enter Information (Auth Server / Basic Info)

Enter institution name, address, application form, and authentication server details.

04.
Register Campus and Contact Information

Enter campus information to be displayed on the eduroam map.

Important Notice

Technical coordination with the Operations Center is required before connecting your authentication server.
